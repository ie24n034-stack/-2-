# シフト管理システム（クライアント01）

飲食チェーン（3店舗・スタッフ40名）向けのシフト自動生成・管理システム。

## 現状と課題
- **状況:** シフトをLINEとExcelで手作業管理。毎週組み直している。
- **問題:** ダブルブッキング、急な欠勤で崩れる、誰が空いているか不明。

## クライアントからの依頼・要件
・「スタッフが空き時間を入力して、自動でかぶりなくシフトが組めるシステムがほしい。」

・「店舗が増えたので、店舗ごとに分けたい。」

### 主な機能（予測されるタスク）
-  スタッフの空き時間（シフト希望）入力機能
-  3店舗分のシフト管理・切り替え機能
-  自動シフト生成アルゴリズム（重複防止）
-  急な欠勤に対応するためのスタッフ空き状況一覧
